irc-qik-bot ~ automated babbling bot

GitHub: https://github.com/abhishekkr/irc-qik-bot/

RubyGems: http://rubygems.org/gems/irc-qik-bot

usage currently require the 'bot nick' to be used; will also pin it to conventional !command

Current Features:

  • basic IRC Network, Channel Connector
  • Help ~ 'qbot: help'
  • Babbler ~ 'qbot: babble'
  • Quote ~ 'qbot: quote'
  • URL Un-Shortener ~ 'qbot: unshort http://SHORT_URL'
  • google ~ 'qbot: google search term'

gem install irc-qik-bot
